Robot Repair

A puzzle game to help a broken robot uncover a secret message, letter by letter


Development: Unity 6, C#


Art: Autodesk Maya 2025, Blender 3.6, Procreate

Robot Repair is a cartoon-stylized puzzle word game. It follows a brief narrative, encouraging the player to uncover meaning with interactive text objects. The development of this game focuses on events, listeners, sound, animation, and input handling. It tackles a primary goal of creating a clear game loop with a strong beginning and end, while also implementing a balanced overall feeling for the player. With a strong system of feedback, reaction, and the ability to try again, this game can be utilized for any puzzle-like experience.
